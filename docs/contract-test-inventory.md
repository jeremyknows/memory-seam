# Contract test inventory

This inventory maps the committed contract tests to the safety boundary each one protects. It is documentation only: it does not authorize live reads, Runtime Registry consumption, service startup, package publication, visibility changes, write custody, reindexing, provider/prod/canary authority, or gate authority.

below cover what's actually installed and exercised by `pytest` from the
